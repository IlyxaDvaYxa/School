GIT flow

GITclone- для того,що б "склонити" репозиторій.

GITpull- для того,що б отримати зміни з репозиторія.

GITcheckout -b branchname- створити нову гілку і перемкнутися на неї.

GITadd-додати зміни,що були зроблені локально.

GITcommit -m"commit text"- підписати їх.

GITpush origin branchname- запушити на git сервер.
